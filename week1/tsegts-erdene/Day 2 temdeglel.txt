Github нь файлын түүх хадгалахад зориулсан программ юм. Уг бичлэгт доорх командуудыг тайлбарласан.

Git init
Түүхийг эхлүүлэх команд

.gitignore
Git руу оруулахгүй файлуудаа бичиж өгнө.

Git add.
Бүх файлыг Staging areа руу нэмнэ. Тухайн файлыг track хийнэ.

Git commit -m “message”
Тайлбар бичнэ.

Git push
Cloud руу Local файлыг явуулна.

Git clone
Тухайн git-ийн бүх түүхийг Local дээрээ хуулна.

Git pull
Шинэ өөрчлөлтүүдийг local руу татаж авна.

Git status
Тухайн git-ийн төлөвийг харуулна.

Git diff
Тухайн файл дээрх өөрчлөлтийг харьцуулж харуулна.

Git checkout -b test
Шинэ branch үүсгэн тийшээгээ шилжинэ.
